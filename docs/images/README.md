# Documentation Images

This directory contains images used in the project README and documentation.

## Required Images

| File | Usage | Dimensions |
|------|-------|------------|
| `banner.png` | README header banner | 1200×300 |
| `architecture.png` | System architecture diagram | 1200×700 |
| `workflow.png` | Workflow execution flow | 1000×600 |
| `desktop_deploy.png` | Desktop deployment overview | 1000×600 |
| `web_api_tunnel.png` | Web API tunnel exposure | 1000×500 |
| `provider_matrix.png` | Provider comparison matrix | 1000×500 |
| `web_console_preview.png` | Web console screenshot | 1200×700 |

## Style Guidelines

- Dark technology theme (deep blue-purple gradient background)
- Clean, minimal design
- No watermarks
- Minimal text; text must be clear and readable
- Consistent visual language across all images
- Suitable for GitHub/Gitee project display

## Generation

These images should be generated using AI image generation tools
(e.g., DALL-E 3, Midjourney) with the prompts provided in the project
documentation.
